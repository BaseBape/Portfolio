# 結 YUI｜自然と、人と、未来をむすぶ

京都を拠点とする架空のデザイン＆クラフトスタジオ「結 YUI」のコーポレートサイトです。  
自然素材・地域共創・職人の手仕事をテーマに、和の余白と有機的なビジュアルを取り入れたランディングページ構成になっています。

## 概要

このHTMLは、単一ファイルで完結する静的Webサイトです。

主なセクションは以下です。

- Hero
- Mission
- Services
- Works
- News
- About
- CEO Message
- Careers
- Contact
- Footer

## 使用技術

- HTML
- CSS
- Vanilla JavaScript
- Google Fonts

外部ライブラリやビルドツールは使用していません。

## フォント

Google Fontsから以下を読み込んでいます。

- Shippori Mincho
- Noto Serif JP
- Noto Sans JP
- Zen Kaku Gothic New
- Cormorant Garamond
- EB Garamond
- Inter
- Manrope

## 主な機能

- レスポンシブ対応
- モバイルメニュー
- スクロール時のナビゲーション変化
- 文字単位のヒーローアニメーション
- IntersectionObserverによるスクロールリビール
- カウントアップアニメーション
- Worksセクションのホバープレビュー
- パララックス表現
- ダークテーマ用CSS変数
